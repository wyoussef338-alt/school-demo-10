# دليل تشغيل بوابة مدرسة الصالحين للغات الذكية 🎓
## EL-SALHEEN School Smart Portal - Quick Setup Guide

أهلاً بك يا بطل! لضمان تشغيل الموقع على جهاز اللاب توب الخاص بك **بشكل سليم وبسرعة فائقة ودون أي شاشة بيضاء**، يرجى اتباع هذه الخطوات السهلة، لأن متصفحات الإنترنت تمنع تشغيل ملفات البرمجة الحديثة (ES Modules) بمجرد الضغط المزدوج المباشر على ملف `index.html`. يجب تشغيل السيرفر المحلي الصغير المخصص للمطورين.

---

### 🛠️ متطلبات التشغيل الأساسية (مرة واحدة فقط)
من فضلك تأكد من تثبيت برنامج **Node.js** على جهاز اللاب توب الخاص بك:
1. قم بتحميل وتثبيت **Node.js** من الموقع الرسمي: [https://nodejs.org](https://nodejs.org/) (اختر النسخة الموصى بها LTS).

---

### 💻 خطوات تشغيل الموقع على اللاب توب:

#### 1. افتح مجلد المشروع (Project Folder)
* قم بفك الضغط عن ملف مجلد المشروع الذي قمت بتنزيله.
* افتح واجهة الأوامر (Terminal أو Command Prompt) داخل هذا المجلد. 
  *(في نظام ويندوز: يمكنك كتابة `cmd` في شريط مسار الملفات بالعلوي داخل المجلد والضغط على Enter).*

#### 2. تثبيت المكتبات البرمجية (Install Dependencies)
اكتب الأمر التالي واضغط Enter لتثبيت جميع الأدوات والمكتبات اللازمة تلقائياً:
```bash
npm install
```
*سيقوم هذا الأمر ببناء مجلد المكونات `node_modules` خلال ثوانٍ قليلة.*

#### 3. تشغيل الموقع في وضع التطوير المحلي (Run Dev Server)
قم بكتابة الأمر التالي لتشغيل السيرفر السحابي المحلي على جهازك:
```bash
npm run dev
```

#### 4. افتح المتصفح واستمتع بالموقع!
بمجرد كتابة الأمر السابق، سيظهر لك مسار العنوان المحلي:
```text
  ➜  Local:   http://localhost:3000/
```
* قم بنسخ العنوان `http://localhost:3000` وافتحه في متصفح كروم أو أي متصفح لديك.
* ستجد الموقع يعمل بكامل جماله، وسلاسته، وخلفياته الساحرة المتحركة كلياً وبأقصى استجابة ممكنة دون أي تعليق!

---

### 🥇 الميزات الرائعة المدمجة حديثاً بالتصميم الفاخر:
- **تصميم ومظهر STEM-Egypt**: خلفيات كلاسيكية لساحات المدرسة، وشريط تنقل زجاجي (Backdrop blur) ينكمش ويتحول بانسيابية عند تمرير الصفحة لأسفل.
- **شعار الصالحين الفيكتور الطبيعي (SVG Crest)**: قمنا برسم درع المدرسة الرسمي رقمياً ليضم شارة العلوم، الرياضة، والتقدم المعرفي.
- **سلايدر وخلفيات تفاعلية**: صور عالية الجودة تتنقل تلقائياً أو عبر الضغط على خطوط المؤشرات السفلية تماماً كالمطلوب في الصورة المرفقة.
- **المساعد الذكي (AI)**: مستكشف ذكي وتفاعلي مع طلاب ومعلمي الصالحين.
- **توافقية كاملة**: سرعة عالية لملفات الجافا سكريبت ولائحة التحكم خالية تماماً من الكراشات.
